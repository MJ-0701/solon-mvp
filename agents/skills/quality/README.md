# 제품품질본부 스킬

## 개요
제품품질본부 전용 업무 프레임워크입니다.

## 사용 에이전트
- **CPO** (Opus) — 평가 기준 설계, 최종 Pass/Fail
- **제품품질 본부장** (Opus) — 테스트 전략, Gap Analysis, 설계 검토
- **QA 엔지니어** (Sonnet) — Gap Analysis 실행, 코드 품질 검사
- **데이터 수집** (Haiku) — 로그/메트릭 수집
