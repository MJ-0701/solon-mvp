# 업무 스킬 체계

## 개요
에이전트 팀의 **업무 수행 프레임워크**를 정의하는 스킬 모음입니다.
프로젝트 루트 `skills/`의 도메인 스킬(비즈니스 지식)과는 별도 체계입니다.

## 구분
| 구분 | 위치 | 목적 |
|------|------|------|
| **업무 스킬** (여기) | `.claude/agents/skills/` | 업무 절차/프레임워크 |
| **도메인 스킬** | `skills/` (프로젝트 루트) | 코드/기능별 비즈니스 지식 |

## 폴더 구조
| 폴더 | 설명 | 대상 |
|-------|------|------|
| `common/` | 전 본부 공통 (PDCA 워크플로우) | 전원 |
| `strategy/` | 전략기획본부 전용 | CEO, 전략기획 본부장, 리서처 |
| `engineering/` | 기술개발본부 전용 | CTO, 기술개발 본부장, 개발자 |
| `quality/` | 제품품질본부 전용 | CPO, 제품품질 본부장, QA 엔지니어 |

## 매트릭스
각 에이전트별 스킬 사용 권한은 `SKILL_MAP.md` 참조 (W=작업 수행, R=참조만).
