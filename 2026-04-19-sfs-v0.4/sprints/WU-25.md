## §1. `/sfs plan` 구현 spec (V-1 conditions #4 적용 — 파일 path stdout 출력만)

### 1.1 Output 형식 확정

```
plan.md ready: .sfs-local/sprints/<sprint-id>/plan.md
```

(에디터 launch 안 함. 사용자가 직접 `vim/code/cursor` 로 open. WU-23 §1.3 #4 사용자 결정 verbatim 정합.)

### 1.2 Bash 골격 (sfs-plan.sh draft)

```sh
#!/usr/bin/env bash
# sfs-plan.sh — /sfs plan dispatch (WU-25 §1)
set -euo pipefail
SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
. "$SCRIPT_DIR/sfs-common.sh"

validate_sfs_local                       # exit 1 if .sfs-local missing
SPRINT_ID="$(read_current_sprint)"       # exit 1 if no active sprint
[ -n "$SPRINT_ID" ] || die 1 "no active sprint, run /sfs start first"

SPRINT_DIR=".sfs-local/sprints/$SPRINT_ID"
PLAN_PATH="$SPRINT_DIR/plan.md"

# (a) plan.md 미존재 시 template 적용 (sprint-templates/plan.md → cp + sed for sprint-id)
if [ ! -f "$PLAN_PATH" ]; then
    TEMPLATE=".sfs-local/sprint-templates/plan.md"
    [ -f "$TEMPLATE" ] || die 4 "template missing: $TEMPLATE"
    cp "$TEMPLATE" "$PLAN_PATH"
    sed -i.bak "s/{{SPRINT_ID}}/$SPRINT_ID/g" "$PLAN_PATH" && rm -f "$PLAN_PATH.bak"
fi

# (b) frontmatter 갱신 — phase: plan + last_touched_at
NOW="$(date -u +%Y-%m-%dT%H:%M:%S+00:00)"
update_frontmatter "$PLAN_PATH" "phase" "plan"
update_frontmatter "$PLAN_PATH" "last_touched_at" "$NOW"

# (c) events.jsonl append
append_event "{\"type\":\"plan_open\",\"sprint_id\":\"$SPRINT_ID\",\"path\":\"$PLAN_PATH\",\"ts\":\"$NOW\"}"

# (d) stdout
echo "plan.md ready: $PLAN_PATH"
```

## §2. `/sfs review` 구현 spec (gate verdict 통합 = gates.md §3 enforce)

### 2.1 Output 형식 확정

```
review.md ready: <path> | gate <gate-id> awaiting verdict
```

(`--gate <id>` 미지정 시 마지막 active gate id 추론 = events.jsonl 의 마지막 `review_open` event 의 `gate_id` 필드. 추론 실패 시 stderr `gate id required: --gate <id>` exit 6.)

### 2.2 Bash 골격 (sfs-review.sh draft)

```sh
#!/usr/bin/env bash
# sfs-review.sh — /sfs review dispatch (WU-25 §2)
set -euo pipefail
SCRIPT_DIR="$(cd "$(dirname "$0")" && pwd)"
. "$SCRIPT_DIR/sfs-common.sh"

GATE_ID=""
while [ $# -gt 0 ]; do
    case "$1" in
        --gate) GATE_ID="$2"; shift 2;;
        --gate=*) GATE_ID="${1#--gate=}"; shift;;
        *) die 7 "unknown arg: $1";;
    esac
done

validate_sfs_local
SPRINT_ID="$(read_current_sprint)"
[ -n "$SPRINT_ID" ] || die 1 "no active sprint, run /sfs start first"

# (a) gate id 검증 (gates.md §1 7-enum exact match, case-sensitive)
if [ -z "$GATE_ID" ]; then
    GATE_ID="$(infer_last_gate_id)"   # events.jsonl tail scan, 없으면 빈 문자열
    [ -n "$GATE_ID" ] || die 6 "gate id required: --gate <id>, valid: G-1, G0, G1, G2, G3, G4, G5"
fi
validate_gate_id "$GATE_ID" || die 6 "unknown gate $GATE_ID, valid: G-1, G0, G1, G2, G3, G4, G5"

SPRINT_DIR=".sfs-local/sprints/$SPRINT_ID"
REVIEW_PATH="$SPRINT_DIR/review.md"

# (b) review.md 미존재 시 template 적용
if [ ! -f "$REVIEW_PATH" ]; then
    TEMPLATE=".sfs-local/sprint-templates/review.md"
    [ -f "$TEMPLATE" ] || die 4 "template missing: $TEMPLATE"
    cp "$TEMPLATE" "$REVIEW_PATH"
    sed -i.bak "s/{{SPRINT_ID}}/$SPRINT_ID/g; s/{{GATE_ID}}/$GATE_ID/g" "$REVIEW_PATH" && rm -f "$REVIEW_PATH.bak"
fi

# (c) frontmatter 갱신
NOW="$(date -u +%Y-%m-%dT%H:%M:%S+00:00)"
update_frontmatter "$REVIEW_PATH" "phase" "review"
update_frontmatter "$REVIEW_PATH" "gate_id" "$GATE_ID"
update_frontmatter "$REVIEW_PATH" "last_touched_at" "$NOW"

# (d) events.jsonl append (verdict 필드는 비움 — WU-26 close 시 채움)
append_event "{\"type\":\"review_open\",\"sprint_id\":\"$SPRINT_ID\",\"gate_id\":\"$GATE_ID\",\"path\":\"$REVIEW_PATH\",\"ts\":\"$NOW\"}"

# (e) stdout
echo "review.md ready: $REVIEW_PATH | gate $GATE_ID awaiting verdict"
```
