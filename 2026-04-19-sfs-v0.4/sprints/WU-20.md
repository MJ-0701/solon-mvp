### Install.sh 핵심 설계

- **Mode 판별**: `${BASH_SOURCE[0]}` 존재 + templates/ 병존 → local / 아니면 remote (git clone temp)
- **TTY 처리**: stdin pipe 면 `exec < /dev/tty` — curl|bash 에서도 대화형 read 가능
- **충돌 처리**: skip (default) / backup (`.bak-YYYYMMDD-HHMMSS`) / overwrite / diff (루프 재진입)
- **멱등성**: .sfs-local/sprints/ + decisions/ 은 merge 모드로 기존 산출물 절대 보존
- **.gitignore 마커**: `### BEGIN solon-mvp ### … ### END solon-mvp ###` — upgrade/uninstall 시 블록 단위 교체
- **VERSION 기록**: consumer 의 `.sfs-local/VERSION` 에 installed_at / mode / source_repo 기록
