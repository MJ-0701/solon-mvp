# WU-40 — Beginner Guide for Non-Developer Onboarding

## §0. Goal

Create a plain-language guide for designers and other non-developer users who
are blocked before the first successful `sfs status`.

## §1. Acceptance Criteria

1. Add a dedicated beginner guide with Windows/Scoop and Mac/Homebrew paths.
2. Explain required concepts without assuming terminal, Git, Scoop, or Homebrew knowledge.
3. Show the success signal and troubleshooting paths for common install failures.
4. Link the guide from README and package manager surfaces.
5. Ensure release tooling includes the new guide in stable product cuts.
