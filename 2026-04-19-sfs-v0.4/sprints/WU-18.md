## 제약 / 규율 준수

- **원칙 2 (self-validation-forbidden)**: 템플릿 placeholder 만 제공. 의미 결정 (stack / ownership / repo name / 현금영수증 API) 은 사용자가 Mac 에서 채움. A/B/C 선택지는 PHASE1-KICKOFF-CHECKLIST 에 이미 적힌 "권장값" 을 그대로 반영 (재해석 금지).
- **IP 경계 엄격**: 모든 템플릿에 Solon 경로 / repo URL 하드코딩 없음. `<PROJECT-NAME>` / `<STACK>` / `<DB>` / `<DEPLOY>` placeholder 만.
- **200-line rule**: WU-18 은 14 신규 파일 예상, 각 파일은 50~100 line 이하. atomic single commit 가능.
- **Option β (minimal)**: 템플릿 최소 골격만 유지. W1 실행 중 사용자가 필요에 따라 확장.
